# Iris Data Analysis Dashboard

## Overview

This project is an interactive Iris Dashboard built using Python and Shiny for Python. The dashboard provides exploratory analysis and visualization of the famous Iris flower dataset, allowing users to examine the characteristics of different Iris species through interactive charts and filters.

## Objectives

* Explore the relationships between flower measurements in the Iris dataset.
* Compare the characteristics of different Iris species.
* Visualize patterns and distributions within the dataset.
* Demonstrate data analysis, visualization, and dashboard development skills.

## Tools and Technologies

* Python
* Pandas
* Plotly Express
* Shiny for Python
* VS Code

## Dataset

The Iris dataset contains measurements of 150 Iris flowers from three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

Features include:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species

## Dashboard Features

* Interactive filtering by Iris species.
* Scatter plots for feature comparison.
* Distribution analysis of flower measurements.
* Summary statistics for selected species.
* Dynamic visualizations for exploratory data analysis.

## Key Insights

* Iris Setosa is clearly distinguishable from the other species based on petal dimensions.
* Petal length and petal width show strong relationships with species classification.
* Iris Versicolor and Iris Virginica share some similarities but can still be differentiated through feature analysis.
* Interactive visualizations make it easier to identify patterns and compare species characteristics.

## How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/adeola-data-science/Iris-Dashboard.git
```

### 2. Navigate to the project folder

```bash
cd Iris-Dashboard
```

### 3. Install required packages

```bash
pip install -r requirements.txt
```

### 4. Run the dashboard

```bash
shiny run --reload app.py
```
